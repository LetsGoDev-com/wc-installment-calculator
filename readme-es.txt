Plugin gratuito para WordPress que muestra opciones de pago a plazos en las páginas de tu tienda WooCommerce.

== Description ==
El plugin toma automáticamente el precio actual del producto y calcula el valor de las cuotas según el número de cuotas y el porcentaje de cargo adicional definido por el administrador. Esto ayuda a los clientes a comprender claramente cuánto pagarían por cuota antes de finalizar la compra.

Sin integración con pasarelas de pago ni modificaciones en el proceso de compra: solo información clara y sencilla sobre las cuotas.

* Calculadora de cuotas en la página del producto
* Añadir varias cuotas
* Múltiples diseños disponibles
* Compatibilidad con WooCommerce HPOS

== Installment Calculator - Features ==
* Muestra la información de pago a plazos en las páginas de producto de WooCommerce
* Usa el precio actual del producto automáticamente
* El administrador puede definir:
* Número de cuotas
* Porcentaje de cargo adicional
* Calcula el importe de las cuotas dinámicamente
* Solo informativo (no afecta el pago ni los totales del pedido)
* Configuración sencilla y fácil de usar
* Ligero y de alto rendimiento
* Compatible con todos los temas de WooCommerce
* No requiere API externas ni servicios de terceros

== Installation ==
1. Descomprime y sube el plugin 'wc-installment-calculator' al directorio '/wp-content/plugins/'
2. Sube el plugin
3. Activa el plugin desde la sección de plugins
